Справка и ссылки

►
JSONC‑комментарии: карточки «Правила/Балансировщики» применяют изменения в редактор точечно и стараются сохранить комментарии рядом с элементами. Для надёжной привязки комментариев при перетаскивании задавайте уникальный ruleTag у правил и tag у балансировщиков. Диагностика: откройте консоль браузера и выполните localStorage.setItem('xk.routing.jsonc.debug','1'), затем повторите действие — причины fallback будут в toast/console.

Порты и Исключения

▼

Файлы из каталога /opt/etc/xkeen:

port_proxying.lst

Порты и диапазоны, которые нужно отправлять через прокси.

Порты, которые нужно исключить из проксирования.

      Справка: файловый менеджер
      
    

    

      

        
Двухпанельный менеджер (как Total Commander): действия применяются к активной панели и текущему выделению.


        
Панели

        
    
          
  • Target: local — файлы на роутере (в пределах разрешённых корней), remote — подключение к серверу (SFTP/FTP/FTPS).
    • 
          
  • Connect…/Disconnect — открыть/закрыть remote-сессию.
    • 
          
  • Поле пути: введите путь и нажмите Enter для перехода.
    • 
          
  • ↑ — перейти в родительский каталог, ⟳ — обновить список.
    • 
        


        
Выделение и навигация

        
    
          
  • Клик — фокус + выделение; двойной клик или Enter — открыть файл/папку.
    • 
          
  • ↑/↓ — перемещение фокуса по списку.
    • 
          
  • Ctrl (или клик по чекбоксу) — добавить/убрать из выделения; Shift — выделить диапазон.
    • 
          
  • ПКМ — контекстное меню (открыть/скачать/копировать/переместить/переименовать/удалить/создать…).
    • 
        


        
Операции

        
    
          
  • ➕ Папка / ➕ Файл — создание в активной панели.
    • 
          
  • ⬆ Upload — загрузка файлов с компьютера в активный каталог.
    • 
          
  • ⬇ Download — скачать выделенное (папка/несколько элементов скачиваются как .zip).
    • 
          
  • F5 — копировать из активной панели в другую, F6 — переместить.
    • 
          
  • Если в назначении есть файлы с теми же именами — появится окно «Конфликты» (заменить/пропустить).
    • 
          
  • F2 — переименовать (только в текущем каталоге), F8 — в корзину (в корзине — удалить навсегда).
    • 
          
  • Drag&Drop: перетаскивание между панелями (по умолчанию move; удерживайте Ctrl для copy) или выбор действия в диалоге.
    • 
          
  • Операции — список фоновых задач (копирование/перемещение/удаление/загрузка) и их прогресс.
    •
